# A股每日行情记录

这是一个用于记录每日A股市场行情的仓库。

## 仓库用途

每天与 **DeepSeek AI** 进行交流，获取当日A股市场行情分析，然后将关键信息整理成表格格式保存下来。

## 内容结构

每个文件以日期命名（如 `2026-04-16.md`），包含：

- **强势板块个股**：当日表现强势的股票，包括股票代码、名称、概念/板块、涨跌幅等信息
- **弱势板块个股**：当日表现较弱的股票信息
- **板块资金流向**：主力资金的流入流出情况

## 更新频率

每日更新，持续记录A股市场行情数据。

## 数据来源

通过 DeepSeek AI 对当日市场行情进行分析和总结，提取关键股票信息制作成表格。
